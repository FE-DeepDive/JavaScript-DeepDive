## 함수 정의

> 함수를 호출하기 이전에 매개변수,실행할 문, 반환 값을 지정하는 것입니다.

함수를 정의하는 방법에는 크게 4가지가 있습니다.

- 함수 선언문
  ```js
  function sum(a, b) {
    return a + b;
  }
  ```
- 함수 표현식
  ```js
  const sum = function (a, b) {
    return a + b;
  };
  ```
- Function 생성자 함수
  ```js
  const sum = new Function("a", "b", "return a+b");
  ```
- 화살표 함수 (ES6)
  ```js
  const sum = (a, b) => a + b;
  ```

하나씩 천천히 살펴보겠습니다.
